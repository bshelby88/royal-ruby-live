## 06 — Mythbust: "Pay-for-delete"

What you actually want is *removal* via dispute if it's inaccurate — or *goodwill deletion* if you can show hardship. Paying first is negotiating from weakness."

**CTA:**

## Production checklist (per batch)

- [ ] Phone on a tripod, eye-level
- [ ] Natural window light, 45° to the face
- [ ] Rode or lav mic (phone mic = algo death)
- [ ] Solid background — bookshelf or clean wall
- [ ] Film all 10 in one shirt change (batching signal)
- [ ] Cut each clip in CapCut — captions auto-generated, then manually corrected
- [ ] Export 1080×1920, 30fps, no watermark
- [ ] Cross-post: TikTok → IG Reels → YT Shorts → FB Reels (same file, different captions)

- Which **hook style** has the highest watch-through rate (TikTok analytics → For You)
- Which **topic** drives the most bio clicks (Vercel analytics → `utm_source=tiktok`)
- Which **post time** wins
- Keep what works, kill what doesn't, batch 10 more
