## Rules for the calendar

1. **Never post the same asset on the same platform twice** in 30 days. Rotate.
2. **If a video breaks out** (>10K views on any platform), pause the calendar and make 3 follow-up videos on the same topic in the next 48 hours.
3. **If a thread breaks out** (>5K impressions), pin it for 24 hours and reply to every good-faith reply.
4. **Every post must have the right tracked short link** in bio or description. Mismatch = lost attribution.
5. **If you miss a day** — skip it. Don't double-post to catch up. Double-posting makes the algorithm angry.

## Adjustments based on early data

After Day 7, check Vercel Analytics:
- If TikTok outperforms Instagram by 3x, shift bandwidth to TikTok (post 2/day there, 0.5/day on IG).
- If a specific video topic is driving checklist signups disproportionately, make 3 more on the same topic.
- If Farcaster Frame mints outperform Stripe conversions, shift NFT-mint focus higher in every caption.

## Automation opt-in (when you're ready)

Once the Blotato API key is wired (already in `.env.local`), you can auto-schedule the entire calendar with:

```bash
node scripts/schedule-posts.mjs --execute
```

That command reads this calendar indirectly through `scripts/schedule-posts.mjs`'s `PLAN` array. Expand `PLAN` with the entries above and it'll fan out to all 4 platforms from one command.
